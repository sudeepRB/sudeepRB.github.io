YoungStar Club Project

A responsive sports club website developed as a freelance project.

Project Overview

This project involved designing and developing a fully responsive website for the YoungStar Club, a sports organization. The main goal was to create an engaging online presence that highlights the club's activities, news, and member information.

Technologies Used

  • HTML5, CSS3, JavaScript
  • Bootstrap, jQuery
  • Adobe Photoshop, Illustrator, XD
  • PHP, MAMP Server

Key Features

  • Responsive design optimized for various devices
  • User-friendly navigation and content layout
  • Custom graphics and illustrations created using Adobe Suite
  • Dynamic content management through PHP integration
