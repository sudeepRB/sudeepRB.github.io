Storyboard
Backend (How it works as a developer): –
At first I created a model of my player(Spaceship) and my hazards(Asteroid) in Autodesk Maya. I also designed the background in illustrator. I downloaded the royalty free audio files required for my project. And once I did that I imported all those to my assets folder in my unity project. Inside unity I created all the required game object as a player, enemy, GUI Text, and background. And I used C# script to make those objects work in real time.
Front-End (How it works as a player): –
It’s a 3D shooting game where spaceship is the player and asteroids are the hazards. User can simply use left, right up down arrow from the keyboard in order to do the movements. To shot the asteroid user has to simple press the ‘spacebar’. Once the game is over user can still restart by pressing ‘R’.
